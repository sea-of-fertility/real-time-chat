== 개요

WebSocket + SSE 기반 1:1 실시간 채팅 백엔드 API 문서입니다.

|===
| 항목 | 값

| Host
| `localhost:8080`

| 인증
| JWT Bearer Token

| Content-Type
| `application/json`
|===
